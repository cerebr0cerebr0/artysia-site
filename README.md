# ARTYSIA Website V6

Premium React + Vite website for ARTYSIA.

## Included

- Full-screen video hero
- Didot-style headings and Inter body font
- Shorter homepage with separate pages
- Collections structure: Tableware, Decor, Signature Pieces, Seasonal Collections
- Bespoke “Create Your Own Piece” page with enquiry form structure
- Journal page with sourcing, behind-the-scenes and spaces content
- Contact page with social icons
- Privacy Policy, Data Policy, Terms and Cookie Policy as footer modals
- Responsive layout for iPhone, Android and tablet
- Vercel security headers in `vercel.json`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
