# ARTYSIA Website

Premium React + Vite website for ARTYSIA.

## Included

- Full-screen video hero
- ARTYSIA color logo
- Didot-style headings with Inter body text
- Collections, Craftsmanship, Bespoke Projects, Hospitality and Journal sections
- Instagram, LinkedIn and Facebook icons
- Privacy Policy, Data Policy, Terms & Conditions and Cookie Policy sections
- Newsletter and contact areas

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
